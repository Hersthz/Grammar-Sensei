Core Settings

AI Cloud

Cloud AI is optional. Grammar Sensei calls only your backend proxy and never stores provider API keys in the extension.

When enabled, the extension sends the current sentence, compact local analysis, grammar DB version, and candidate grammar IDs. It does not send full page text, page URL, or page title.

Privacy & Domains

Grammar Sensei stays local by default. Cloud AI runs only when AI mode is Cloud, AI privacy consent is accepted, a backend endpoint is configured, and the user clicks Ask AI.

Data Management

Export includes settings, history, and notebook. Import replaces those local records.

Grammar Sensei Pro

Free

Pro unlocks Anki/CSV export, unlimited page scan, notebook sync, and advanced stats ($10 one-time). This toggle is a local testing switch until real payments are connected.
